# 🛍️ Customer Behaviour Dashboard
### 📊 End-to-End Data Analytics Project using Python, SQL, and Power BI
## 📊 Project Overview

The **Customer Behaviour Dashboard** is an end-to-end **data analytics project** designed to analyze customer shopping behavior and generate meaningful business insights using **Python, SQL, and Power BI**.

This project focuses on understanding customer spending patterns, purchase frequency, discount usage, subscription impact, and product performance. It follows a complete analytics workflow starting from **data cleaning and preprocessing** to **SQL analysis** and finally **interactive dashboard visualization**.

---

## 🎯 Objectives

- Analyze customer purchase behavior
- Identify revenue trends across different customer segments
- Measure the impact of discounts and subscriptions on spending
- Segment customers based on purchase history
- Evaluate product performance using ratings and sales trends
- Build a complete end-to-end analytics workflow

## 🔄 Project Workflow

### 🧹 1. Data Cleaning and Preprocessing
The raw dataset was cleaned and prepared using **Python** and **Pandas**. The key preprocessing steps included:

- Loading the dataset into a Pandas DataFrame
- Checking data types and missing values
- Handling missing values in the `review_rating` column using median imputation based on product category
- Standardizing column names into `snake_case`
- Removing redundant columns after validation

These steps ensured the dataset was clean, consistent, and ready for analysis.

### ⚙️ 2. Feature Engineering
New features were created to improve the analytical depth of the project:

- **Age Group:** Customers were grouped into Young Adult, Adult, Middle-aged, and Senior using quartile-based binning
- **Purchase Frequency Days:** Purchase frequency categories were converted into numerical day values

### 🗄️ 3. SQL Analysis
SQL queries were written on **PostgreSQL** to answer important business questions such as:

- Revenue generated by male and female customers
- High-spending customers who used discounts
- Top products based on average review ratings
- Comparison of purchase amounts by shipping type
- Spending behavior of subscribed versus non-subscribed customers
- Products with the highest percentage of discounted purchases
- Customer segmentation into New, Returning, and Loyal
- Top 3 most purchased products within each category
- Subscription behavior of repeat buyers
- Revenue contribution by different age groups

### 🧠 Advanced SQL Concepts Used
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- CASE Statements

### 📊 4. Dashboard and Visualization
An interactive **Power BI dashboard** was created to present the findings in a clear and visually appealing way.

The dashboard includes:
- Revenue distribution by gender and age group
- Customer segmentation analysis
- Product performance insights
- Subscription and discount impact analysis
